# Week 1 Task 2 — Shell Scripting & System Monitoring

## Scenario
Manager requested automated server health monitoring scripts.

## Scripts Created
- system-info.sh — full system report
- health-check.sh — checks disk, memory, CPU with warnings
- log-report.sh — saves daily reports to file

## What I learned
- Writing bash scripts with variables and conditions
- Using top, free, df commands to monitor system resources
- if/else statements in bash
- Automating scripts with cron jobs
- Saving output to log files

## How to run
chmod +x *.sh
./system-info.sh
./health-check.sh
./log-report.sh
